# Menú de métodos para generar el CSV de abogados (prov. Ecuador)\n\n## Objetivo\nGenerar un CSV con contactos (teléfono/dirección/site/link) para luego enviarlo a Meta Ads / WhatsApp desde `mensajeria/`.\n\nEl pipeline existente en el repo es:\n1) **Extracción** (obtener CSV bruto)\n2) **Limpieza/normalización**: `Scrapling/scripts/limpiar_csv.py`\n3) **Envío/campaña**: `mensajeria/whatsapp_campaign_from_csv.py` (o el conector Meta)\n\n---\n\n## Método 1 — MCP de Claude + Scrapling (el “más avanzado”)\n**Dependencia:** Scrapling MCP en Claude Desktop/Code + (salvo bloqueo) acceso a Google Maps.\n\n**Ventajas:**\n- Calidad potencialmente alta (datos ricos).\n- Menos trabajo manual de scraping.\n\n**Desventajas:**\n- Requiere disponibilidad de MCP / saldo de la herramienta (según tu escenario).\n- Puede fallar por bloqueos de Google Maps.\n\n**Cuándo usarlo:**\n- Cuando tengas saldo o disponibilidad en el momento y quieras extraer la lista más completa.\n\n---\n\n## Método 2 — Places API (legal/oficial)\n**Dependencia:** Google Places API (tiene costo, es estable y legal).\n\n**Ventajas:**\n- Más estable que scraping.\n- Menos frágil ante cambios de UI.\n\n**Desventajas:**\n- Requiere costo.\n- Para teléfonos/direcciones puede depender de disponibilidad del campo.\n\n**Cuándo usarlo:**\n- Cuando haya presupuesto y quieras algo robusto a futuro sin depender de MCP.\n\n---\n\n## Método 3 — Scraping de respaldo (navegador)\n**Dependencia:** automatización con navegador (p. ej. Playwright/Selenium) hacia fuentes públicas.\n\n**Ventajas:**\n- Puedes evitar APIs de pago si encuentras fuentes públicas suficientes.\n\n**Desventajas:**\n- Frágil: bloqueos, cambios de HTML/JS.\n- Riesgo de perder calidad o requiere mantenimiento.\n\n**Cuándo usarlo:**\n- Cuando no haya saldo para MCP, pero necesites una vía temporal para obtener un CSV.\n\n---\n\n## Cómo escoger (regla simple)\n- Si tienes saldo/disponibilidad para MCP: **Método 1**.\n- Si tienes presupuesto y quieres estabilidad: **Método 2**.\n- Si estás sin saldo y necesitas “salvar” la lista temporalmente: **Método 3**.\n\n---\n\n## Nota de compatibilidad con el repo\nIndependientemente del método, el **CSV bruto** debe convertirse en el **CSV limpio** con `Scrapling/scripts/limpiar_csv.py`.\nAjustaremos un “adapter” si el CSV bruto trae nombres de columnas diferentes.\n
